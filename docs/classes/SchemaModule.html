createTable

  • createTable(table: string): CreateTableBuilder
    • Defined in src/schema/schema.ts:48

    Create a new table.

    example

    This example creates a new table with columns id, first_name, last_name and gender:

    await db.schema
      .createTable('person')
      .addColumn('id', 'int', col => col.primaryKey().increments())
      .addColumn('first_name', 'varchar', col => col.notNull())
      .addColumn('last_name', 'varchar', col => col.notNull())
      .addColumn('gender', 'varchar')
      .execute()
